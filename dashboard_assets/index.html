About

This is a real-time LED Matrix system built on a Raspberry Pi, designed to display dynamic content across multiple panels.

The system rotates between categories like Pokémon, weather, jokes, and science facts, while also allowing live interaction through a web dashboard.

Admins can control the display, manage runtime behavior, and override the screen with custom messages when needed.

Designed and built by coayo.
